
🚀 Exciting 5th Week in the @Webzenith Solutions 100 Days Bootcamp! 🎉

This week, I successfully completed my Week 5 project: WeatherNow - A Weather App with Context API.

Through this project, I learned how to use advanced React concepts such as useState, useEffect, useContext, Context API, and custom hooks. I also understood how to manage global theme state, fetch real-time weather data from a public API, and handle loading and error states professionally.

The application allows users to search for any city and view live weather details including temperature, weather condition, humidity, wind speed, and weather icons. I also implemented a dark/light mode feature using Context API.

This project helped me improve my React project structure, reusable component creation, API handling, and UI development skills.
